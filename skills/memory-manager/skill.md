# 记忆管理 Skill

当用户涉及记忆相关操作时，使用此 Skill 指导行为。

## 记忆查询

当用户询问"之前说过什么"、"你还记得吗"、"我提过什么"等问题时：

1. 调用 `memory_search` 工具搜索相关记忆
2. 将检索到的记忆自然整合到回答中
3. 如果没有找到相关记忆，坦诚告知用户

## 记忆保存

当用户说"记住这个"、"别忘了"、"保存一下"时：

1. 调用 `memory_save` 工具保存记忆
2. 根据内容判断记忆类型：
   - 事件经历、对话要点 → `episodic`
   - 事实知识、用户偏好、个人信息 → `semantic`
   - 操作流程、方法步骤 → `procedural`
3. 评估重要度（0-1）：
   - 核心偏好/关键事实 → 0.8-1.0
   - 一般信息 → 0.5-0.7
   - 次要细节 → 0.3-0.5

## 记忆整理

当用户询问"我有哪些记忆"时：

1. 调用 `memory_search` 搜索相关主题
2. 按类型分类展示记忆（事件/知识/流程）
3. 必要时说明记忆间的关联关系
